NRG 948 Scouting API

Main Site Login Register

An API built for NRG's scouting app's API and to wrap around the Blue Alliance API so we can save users the hassle of stripping unneccesary data

Methods Available

Note: all requests take 'API-Key' in the request. I am working on using temporary tokens.

  • /teams POST - Takes in 'year' and returns active teams.
  • /events POST - Takes in 'year' and returns the event name and keys for all events that year.
  • /event/teams POST - Takes in an 'event_key' and returns teams at the event.
  • /event/matches POST - Takes in an 'event_key', 'comp_level', 'uses_sets' and returns teams on each.
